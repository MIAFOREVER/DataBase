## 系统的基本说明
### 系统简介
这是一个键值对数据库系统，能够实现基本的插入，查询，更新，删除操作。并且与Google的levelDB进行性能对比。
### 系统文件结构说明
```
project
├── gtest
├── include
│   ├── fptree
│   │   └── fptree.h
│   └── utility
│       ├── clhash.h
│       ├── p_allocator.h
│       └── utility.h
├── README.md
├── src
│   ├── bin
│   ├── clhash.c
│   ├── fptree.cpp
│   ├── lycsb.cpp
│   ├── main.cpp

│   ├── utility.cpp
│   └── ycsb.cpp
├── test
│   ├── bin
│   ├── fptree_test.cpp
│   ├── makefile
│   └── utility_test.cpp
└── workloads
```
### 编译
- 编译`src`目录下的文件
```
cd src
make all
```
- 编译`test`目录下的文件
```
cd test
mkdir bin
make all
```
